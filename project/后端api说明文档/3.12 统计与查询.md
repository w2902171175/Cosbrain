## **3.12 统计与查询**

### **3.12.1 `GET /users/count` 获取用户总数**

* **摘要**: 获取平台上注册用户的总数。这是一项基本的用户统计功能。
* **权限**: 已认证用户。（**备注**：在实际生产环境中，此类敏感统计数据接口的访问权限通常会严格限制为系统管理员）。
* **请求体**: 无。
* **响应体**: `schemas.CountResponse`
 * `count` (int): 注册用户的总数量。
* **常见状态码**: `200 OK` (成功获取), `401 Unauthorized` (未认证)
